Random DNA Generation: Generates sequences of custom length (up to 100,000 bp).

FASTA Export: Saves sequences with proper headers and 80-character line wrapping.

Nucleotide Statistics: Calculates percentage distribution of A, C, G, T and total GC-content.

Watermarking: Inserts a user-defined string (e.g., name) into a random position for identification purposes.

Motif Search: Finds all occurrences of a user-provided motif (1-based indexing).

Sequence Complementarity: Generates both complementary and reverse-complementary strands.

In silico Transcription: Converts DNA sequences into mRNA (T → U).

Translation: Translates DNA into amino acid sequences using the standard genetic code.

The program generates a .fasta file named after the provided Sequence ID
